Water Quality AI Pro

This is a machine learning based web application built with Streamlit to predict water quality and determine whether water is safe for drinking.

The system takes key water parameters such as pH, hardness, solids, chloramines, sulfate, conductivity, organic carbon, trihalomethanes, and turbidity as input. It then analyzes the data using a trained machine learning model and rule-based logic to classify water as safe or unsafe.

The project also includes:
- An interactive dashboard to visualize water quality trends
- A prediction tool for real-time water quality analysis
- An AI chatbot that explains water quality concepts in simple language
- Bulk CSV upload for batch predictions
- Data visualizations using Plotly for better insights

The goal of this project is to help users understand water quality easily using artificial intelligence and simple visual tools. It can be useful for learning, research, and basic environmental monitoring projects.

The application is built using:
Streamlit for the web interface
Scikit-learn for machine learning
Pandas and NumPy for data processing
Plotly for visualization
Joblib for model saving and loading

To run the project locally, install the required dependencies and run:
streamlit run app.py

This project demonstrates how machine learning can be applied to environmental data in a simple and interactive way.
